🌍 Traveling Data Analysis Project (Google Sheets)
📌 Project Overview

This project focuses on analyzing traveling data using Google Sheets. The goal is to understand travel patterns, expenses, destinations, and customer behavior through data cleaning, pivot tables, charts, and dashboards.

The analysis helps in identifying:

Most visited destinations

Total travel expenses

Preferred travel modes

Monthly travel trends

Customer booking patterns

📊 Tools Used

Google Sheets – Data cleaning and analysis

Pivot Tables – Data summarization

Charts (Bar, Pie, Line) – Data visualization

Basic Formulas – SUM, AVERAGE, COUNT, IF, etc.

📁 Dataset Description

The dataset includes the following columns:

Booking ID

Customer Name

Destination

Travel Date

Transport Mode (Flight/Train/Bus)

Hotel Cost

Travel Cost

Total Cost

Payment Mode

Travel Status
